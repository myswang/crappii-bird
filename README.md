# crappi-bird

<img src="icon.png" alt="screenshot" width=400>

*A crappy version of that famous mobile game, for Wii*

## Installation

Download the .zip file from the [Releases](https://github.com/myswang/crappi-bird/releases) page. Then extract and copy the files to the root of your SD card, just like any other homebrew application.

## Building from source

You need to install **devkitPro** (Wii homebrew toolchain) and **GRRLIB** (graphics library). I recommend using the documentation from the links below to setup your system correctly:

[Getting Started with devkitPro](https://devkitpro.org/wiki/Getting_Started)

[GRRLIB repository](https://github.com/GRRLIB/GRRLIB)

You can then compile the project with `make`, which will produce a `crappii-bird.dol` binary that you can run on a Wii or Dolphin emulator.
